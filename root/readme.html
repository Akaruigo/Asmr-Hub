安装说明

1.安卓用户下载“QQ浏览器”，找到解压目录打开Asmr-Start.html打开即可，可将此页面设为书签并发送到桌面。
2.IOS用户需要下载“File Hub”，将压缩包导入该软件后点击解压，打开目录中的Asmr-Start.html即可。
3.由于系统差异，IOS上部分图片可能无法正常加载。（从29页开始恢复正常，安卓可能也有部分故障）。

操作说明

1.顺序浏览时每页20个视频。
2.点击卡片右下角的名字可以查看主播更多视频，括号内为资源数量。
3.点击首页上方logo可快速进入美女图片网。
4.左上角“收录资源统计”页列出了所有视频主播统计数据，点击名字可以进入主播分页。
5.分页下方“ASMR1、2、3、”为三个扩展的ASMR资源网站
